# Goal

Solve the following knights-and-knaves puzzle.

## Premises

Twenty islanders, numbered P1 through P20. Each is either a **knight** (who always tells the truth) or a **knave** (who always lies). They make the following statements:

- P1:  "P2 and P3 are of different types."
- P2:  "P3 is a knight."
- P3:  "P1 is a knight."
- P4:  "P3 is a knave."
- P5:  "P4 is a knight."
- P6:  "P5 is a knight."
- P7:  "P6 is a knight."
- P8:  "P7 is a knight."
- P9:  "P8 is a knight."
- P10: "P9 is a knight."
- P11: "P10 is a knave."
- P12: "P11 is a knight."
- P13: "P12 is a knight."
- P14: "P13 is a knight."
- P15: "P14 is a knight."
- P16: "P15 is a knight."
- P17: "P16 is a knight."
- P18: "P17 is a knave."
- P19: "P18 is a knight."
- P20: "P19 is a knight."

## Task

Determine the type (knight or knave) of each islander. Return your final assignment as exactly twenty lines, one per person, in this form:

    P1:  <knight|knave>
    P2:  <knight|knave>
    P3:  <knight|knave>
